# apt: Reference

```bash
# List all installed packages
apt list --installed

# Install a package
apt install 𝑝𝑘𝑔_𝑛𝑎𝑚𝑒

# Install a package without confirmation prompt
apt install 𝑝𝑘𝑔_𝑛𝑎𝑚𝑒 -y
```
